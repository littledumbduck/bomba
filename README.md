# JavaScript Test — Starter Project

This is a minimal HTML/CSS/JS starter scaffold created for quick front-end experiments.

Files created:
- `index.html` — main page
- `css/styles.css` — styles
- `js/main.js` — small interactive demo (add/remove items, theme toggle)

Quick start

1. Open `index.html` in your browser (double-click or `File → Open`).

Or run a simple local server (recommended for fetches or modules):

PowerShell examples:

```
# using Python (if installed)
python -m http.server 3000

# using npx http-server (Node.js required)
npx http-server -c-1 . -p 3000
```

Then open `http://localhost:3000` in your browser.
